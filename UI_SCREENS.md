# واجهات المستخدم الرئيسية (MVP)

## 1. شاشات التسجيل والدخول

### شاشة البداية

1. **العناصر الرئيسية**

   - شعار التطبيق وترحيب.
   - زر تسجيل جديد.
   - زر تسجيل الدخول.
   - اختيار نوع الحساب (ولي أمر/معلم).

2. **الميزات**

   - تحويل سلس بين الشاشات
   - دعم اللغة العربية
   - تصميم جذاب

### تسجيل حساب جديد

1. **حقول الإدخال**

   - نموذج للوالدين:
     - الاسم الكامل.
     - البريد الإلكتروني.
     - كلمة المرور.

   - نموذج للمعلمين:
     - الاسم الكامل.
     - التخصص.
     - البريد الإلكتروني.
     - كلمة المرور.

2. **التحقق**

   - التحقق من صحة البريد
   - قوة كلمة المرور
   - الشروط والأحكام

### تسجيل الدخول

1. **حقول الإدخال**

   - البريد الإلكتروني.
   - كلمة المرور.

2. **التحقق**

   - التحقق من صحة البريد
   - قوة كلمة المرور

## 2. الشاشة الرئيسية

### لوحة التحكم

1. **العناصر الأساسية**

   - قائمة الأطفال.
   - زر "إضافة طفل جديد".
   - التقييمات الحالية.
   - التواصل مع المتخصص.

2. **الإحصائيات**

   - رسوم بيانية للتقدم
   - آخر التقييمات
   - المواعيد القادمة

### القائمة الجانبية

1. **روابط التنقل**

   - الملف الشخصي
   - التقييمات
   - التقارير
   - الإعدادات

2. **معلومات المستخدم**

   - الصورة الشخصية
   - الاسم والدور
   - حالة الاتصال

## 3. إدارة التقييم

### بدء التقييم

1. **اختيار التقييم**

   - قائمة أنواع التقييم
   - معلومات كل نوع
   - خطوات التقييم

2. **إعداد التقييم**

   - اختيار الطفل
   - تحديد المعايير
   - تعليمات البدء

### إجراء التقييم

1. **عرض الأسئلة**

   - عرض أسئلة مقياس كونرز.
   - خيارات الإجابة (0-3).
   - شريط تقدم.

2. **التنقل**

   - زر التالي
   - زر السابق
   - حفظ مؤقت

## 4. شاشات النتائج

### عرض النتائج

1. **ملخص التقييم**

   - جدول الدرجات الخام.
   - تحليل النتائج (درجات تائية).
   - نصوص تفسيرية.
   - توصيات.

2. **التحليل المفصل**

   - تحليل كل مجال
   - الرسوم البيانية
   - الاتجاهات

### التوصيات

1. **عرض التوصيات**

   - توصيات مخصصة
   - خطوات العمل
   - موارد إضافية

2. **خطة المتابعة**

   - جدول زمني
   - تذكيرات
   - مواعيد المراجعة

## 5. شاشات التقارير

### إنشاء التقارير

1. **اختيار نوع التقرير**

   - تقرير تفصيلي
   - ملخص التقدم
   - تقرير مقارنة

2. **تخصيص التقرير**

   - اختيار المحتوى
   - تنسيق العرض
   - خيارات التصدير

### مشاركة التقارير

1. **خيارات المشاركة**

   - مشاركة مباشرة
   - إرسال بالبريد
   - حفظ كملف

2. **إدارة الصلاحيات**

   - تحديد المستلمين
   - مستوى الوصول
   - مدة المشاركة
