# واجهات المستخدم الرئيسية

## 1. شاشات التسجيل والدخول

### شاشة البداية
- شعار التطبيق وترحيب
- زر تسجيل جديد
- زر تسجيل دخول
- اختيار نوع الحساب (والد/متخصص)

### تسجيل حساب جديد
- نموذج تسجيل للوالدين
  * الاسم الكامل
  * البريد الإلكتروني
  * رقم الهاتف
  * كلمة المرور
  * تأكيد كلمة المرور

- نموذج تسجيل للمتخصصين
  * الاسم الكامل
  * التخصص والمؤهلات
  * البريد الإلكتروني
  * رقم الهاتف
  * كلمة المرور
  * تأكيد كلمة المرور

### تسجيل الدخول
- حقل البريد الإلكتروني
- حقل كلمة المرور
- زر نسيت كلمة المرور
- زر تسجيل الدخول

## 2. الشاشة الرئيسية

### واجهة الوالدين
- قائمة الأطفال
- زر إضافة طفل جديد
- التقييمات الحالية
- المواعيد القادمة
- الإشعارات والتنبيهات
- التواصل مع المتخصص

### واجهة المتخصصين
- قائمة الحالات النشطة
- التقييمات الجديدة
- المواعيد اليومية
- الإشعارات والرسائل
- إحصائيات وتقارير

## 3. إدارة الملف الشخصي

### ملف الوالدين
- معلومات شخصية
- قائمة الأطفال
- المتخصصين المرتبطين
- سجل التقييمات
- الإعدادات

### ملف المتخصص
- المعلومات المهنية
- جدول المواعيد
- قائمة الحالات
- التقارير والإحصائيات
- الإعدادات

## 4. إدارة الأطفال (للوالدين)

### إضافة طفل
- المعلومات الأساسية
- التاريخ الطبي
- معلومات المدرسة
- اختيار متخصص

### عرض ملف الطفل
- المعلومات الأساسية
- سجل التقييمات
- خطة العلاج الحالية
- تقدم الأنشطة
- التقارير والملاحظات

## 5. التقييم والمتابعة

### إجراء التقييم
- اختيار نوع التقييم
- أسئلة مقياس كونرز
- خيارات الإجابة
- حفظ وإرسال

### عرض النتائج
- درجات التقييم
- تحليل بياني
- مقارنة بالتقييمات السابقة
- توصيات المتخصص

## 6. نظام الإشعارات والرسائل

### شاشة الإشعارات
- قائمة الإشعارات غير المقروءة
- قائمة الإشعارات المقروءة
- فلترة حسب النوع:
  * تحديثات التقييم
  * رسائل جديدة
  * مواعيد
  * تذكيرات خطة العلاج
- خيارات:
  * تحديد كمقروء
  * حذف
  * تفعيل/إيقاف نوع معين

### شاشة المحادثات
- قائمة المحادثات النشطة
- البحث عن محادثة
- شاشة المحادثة:
  * عرض الرسائل
  * إرسال رسالة نصية
  * مشاركة تقارير وملفات
  * عرض حالة القراءة
  * إمكانية حذف الرسائل

### إعدادات الإشعارات
- تفعيل/إيقاف الإشعارات
- اختيار أنواع الإشعارات
- تخصيص نغمات التنبيه
- جدولة وقت الإشعارات
- إعدادات عدم الإزعاج

## 7. خطة العلاج

### عرض الخطة
- الأهداف العلاجية
- الأنشطة اليومية
- جدول المتابعة
- ملاحظات المتخصص

### تتبع التقدم

- تحديثات الحالة
- التحديات والنجاحات
- تقارير التقدم

## 8. التواصل

### الرسائل
- قائمة المحادثات
- إرسال رسالة جديدة
- عرض المحادثة
- المرفقات والصور

### المواعيد

- حجز موعد جديد
- تعديل المواعيد
- التذكيرات

## 9. التقارير والإحصائيات

### تقارير الوالدين
- تقدم الطفل

- تقارير الأنشطة
- توصيات المتخصص

### تقارير المتخصصين

- تحليل البيانات
- أداء الخطط العلاجية
