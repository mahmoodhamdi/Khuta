# واجهات المستخدم الرئيسية (MVP)

## 1. شاشات التسجيل والدخول

### شاشة البداية
- شعار التطبيق وترحيب.
- زر تسجيل جديد.
- زر تسجيل الدخول.
- اختيار نوع الحساب (ولي أمر/معلم).

### تسجيل حساب جديد
- نموذج للوالدين:
  - الاسم الكامل.
  - البريد الإلكتروني.
  - كلمة المرور.

- نموذج للمعلمين:
  - الاسم الكامل.
  - التخصص.
  - البريد الإلكتروني.
  - كلمة المرور.

### تسجيل الدخول
- البريد الإلكتروني.
- كلمة المرور.
- زر تسجيل الدخول.

## 2. الشاشة الرئيسية

### واجهة الوالدين
- قائمة الأطفال.
- زر "إضافة طفل جديد".
- التقييمات الحالية.
- التواصل مع المتخصص.

### واجهة المعلمين
- قائمة الأطفال.
- التقييمات الجديدة.
- التوصيات.

## 3. إدارة التقييم

### شاشة إجراء التقييم
- عرض أسئلة مقياس كونرز.
- خيارات الإجابة (0-3).
- شريط تقدم.

### عرض النتائج
- جدول الدرجات الخام.
- تحليل النتائج (درجات تائية).
- نصوص تفسيرية.
- توصيات.
