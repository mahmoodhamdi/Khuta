# واجهات المستخدم الرئيسية (MVP)

## 1. شاشات التسجيل والدخول

### شاشة البداية
- شعار التطبيق وترحيب
- زر تسجيل جديد
- زر تسجيل دخول
- اختيار نوع الحساب (والد/متخصص)

### تسجيل حساب جديد
- نموذج تسجيل للوالدين
  - الاسم الكامل
  - البريد الإلكتروني
  - كلمة المرور

- نموذج تسجيل للمتخصصين
  - الاسم الكامل
  - التخصص
  - البريد الإلكتروني
  - كلمة المرور

### تسجيل الدخول
- البريد الإلكتروني
- كلمة المرور
- زر تسجيل الدخول

## 2. الشاشة الرئيسية

### واجهة الوالدين
- قائمة الأطفال
- زر إضافة طفل جديد
- التقييمات الحالية
- التواصل مع المتخصص

### واجهة المتخصصين
- قائمة الحالات النشطة
- التقييمات الجديدة
- الرسائل الواردة

## 3. إدارة الأطفال

### إضافة طفل
- الاسم
- العمر
- الجنس
- ملاحظات أساسية

### عرض ملف الطفل
- المعلومات الأساسية
- سجل التقييمات
- خطة العلاج الحالية

## 4. التقييم

### إجراء التقييم
- مقياس كونرز المبسط
- خيارات الإجابة
- حفظ النتائج

### عرض النتائج
- درجات التقييم
- توصيات أساسية

## 5. التواصل

### الرسائل
- قائمة المحادثات
- إرسال رسالة نصية
- عرض المحادثة
