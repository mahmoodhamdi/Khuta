## 1. نظرة عامة | Overview

يحدد هذا المستند نطاق مشروع تطبيق ADHD للكشف المبكر ومتابعة الحالات.

## 2. النطاق الأساسي (MVP) | MVP Scope

### 2.1 الوظائف الأساسية | Core Functions

1. **نظام التقييم**
   * تسجيل المستخدم
   * إجراء التقييم
   * عرض النتائج
   * تقديم التوصيات

2. **إدارة المستخدمين**
   * تسجيل الحساب
   * إدارة الملف الشخصي
   * إدارة معلومات الأطفال
   * إعدادات الخصوصية

3. **نظام التوصيات**
   * توصيات الأنشطة العلاجية
   * قائمة الأطباء المقترحين
   * نصائح وإرشادات
   * موارد تعليمية

### 2.2 الواجهات | Interfaces

1. **واجهات المستخدم**
   * شاشات التسجيل (5)
   * شاشات التقييم (8)

   * شاشات التوصيات (6)

2. **واجهات النظام**
   * واجهة قاعدة البيانات
   * واجهة التخزين السحابي
   * واجهة المصادقة
   * واجهة التحليلات
