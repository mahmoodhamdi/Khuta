# تطبيق "خُطى" - ADHD Assessment Application

## نظرة عامة

تطبيق "خُطى" هو منصة متكاملة تساعد الوالدين والمتخصصين في تقييم وإدارة اضطراب فرط الحركة ونقص الانتباه (ADHD) لدى الأطفال باستخدام مقياس كونرز. يوفر التطبيق حلاً شاملاً للتقييم وتحليل النتائج والتخطيط العلاجي، مع الحفاظ على خصوصية المستخدمين من خلال التخزين المحلي للبيانات.

## المميزات الرئيسية

1. **تقييم ADHD**
   - تنفيذ مقياس كونرز بشكل رقمي
   - واجهة استبيان سهلة الاستخدام للوالدين
   - تخزين آمن للبيانات
   - حساب النتائج فوري

2. **تحليل النتائج**
   - تفصيل شامل للدرجات
   - تمثيل مرئي للنتائج
   - تتبع البيانات التاريخية
   - مراقبة التقدم المستمر

3. **خطة العلاج**
   - توصيات علاجية مخصصة من المتخصصين
   - تخطيط وجدولة الأنشطة
   - استراتيجيات إدارة السلوك
   - أدوات تتبع التقدم

4. **التواصل والمتابعة**
   - تواصل مباشر بين الوالدين والمتخصصين
   - مشاركة التقارير والنتائج
   - جدولة المتابعات
   - تحديثات مستمرة للخطة العلاجية

## المواصفات التقنية

- **إطار العمل**: Flutter
- **الهيكل**: تخزين محلي أولاً
- **تخزين البيانات**: SQLite مع مزامنة Firebase
- **واجهة المستخدم**: تصميم Material مع دعم العربية
- **الأمان**: تشفير البيانات وحماية الخصوصية

## الخصوصية والأمان

- تخزين البيانات محلياً على الجهاز
- مزامنة آمنة مع Firebase
- تشفير البيانات الحساسة
- نسخ احتياطي دوري

## المستخدمون المستهدفون

- والدي الأطفال المصابين أو المشتبه بإصابتهم بـ ADHD
- المتخصصون النفسيون والسلوكيون
- معالجو النطق والتخاطب
- المعالجون السلوكيون

## التأثير المتوقع

- تشخيص مبكر لأعراض ADHD
- نهج منظم لإدارة ADHD
- تحسين التواصل بين الوالدين والمتخصصين
- نتائج علاجية أفضل من خلال المتابعة المستمرة
- توثيق منظم لتطور الحالة
