# **خُطى - تطبيق تقييم وإدارة ADHD**  

## **نظرة عامة**  

"خُطى" هو تطبيق رقمي مبتكر يساعد في الكشف المبكر عن اضطراب فرط الحركة ونقص الانتباه (ADHD) لدى الأطفال، باستخدام مقياس كونرز المعتمد عالميًا. يوفر التطبيق توصيات مخصصة، خطة علاجية، وقائمة بالأطباء المتخصصين، مما يتيح للآباء والمعلمين متابعة حالة الطفل بفعالية.  

---

## **الميزات الرئيسية**  

- **مقياس كونرز الرقمي**:  
  - استبيان مخصص للوالدين والمعلمين لتقييم الأعراض.  
  - تحليل النتائج باستخدام الدرجات الخام والتائية.  

- **توصيات مخصصة**:  
  - اقتراح أنشطة علاجية.  
  - عرض قائمة بالأطباء المتخصصين بناءً على نتائج التقييم.  

- **إدارة سهلة للحالة**:  
  - متابعة دورية للحالة من خلال تقارير مرئية.  
  - إمكانية إعادة التقييم وجدولة المواعيد.  

- **سهولة الوصول**:  
  - تسجيل الدخول باستخدام البريد الإلكتروني أو حساب Google.  
  - واجهة مستخدم باللغة العربية متوافقة مع جميع الأجهزة.

## **المتطلبات | Requirements**

### **تطبيق فلاتر | Flutter App**
- Flutter SDK v3.16.0+
- Dart SDK v3.0.0+
- Android Studio / VS Code

### **الخلفية | Backend**
- Node.js v18+
- MongoDB v6+
- Firebase Admin SDK

## **التقنيات المستخدمة**  

### **Frontend**  

- **Flutter**: لتطوير واجهات المستخدم المتوافقة مع iOS وAndroid.  

### **Backend**  

- **Node.js وExpress**: لإنشاء واجهة برمجة التطبيقات (API).  

### **Database**  

- **SQLite**: لتخزين البيانات محليًا.  
- **Firebase**: لتخزين البيانات السحابية وإدارة المستخدمين.

## **رحلة المستخدم**  

1. **التسجيل**:  
   - إنشاء حساب جديد باستخدام البريد الإلكتروني أو Google.  
   - اختيار نوع المستخدم (ولي أمر/متخصص).  

2. **إجراء التقييم**:  
   - ملء استبيان مقياس كونرز للطفل.  
   - تحليل تلقائي للنتائج وعرض الدرجات.  

3. **عرض النتائج والتوصيات**:  
   - توصيات بناءً على تحليل النتائج.  
   - عرض قائمة بالأطباء المتخصصين.  

4. **المتابعة**:  
   - جدولة تقييمات دورية.  
   - متابعة التقدم من خلال تقارير مرئية.
