# KHUTA - ADHD Early Detection App  

KHUTA is a mobile application designed to assist in the **early detection of ADHD** in children. It leverages the **Conners' Rating Scale**, providing an easy-to-use tool for parents and teachers to assess symptoms before seeking professional help.

## 🚀 Getting Started  

### 1️⃣ Clone the repository  

```sh
git clone https://github.com/mahmoodhamdi/khuta.git
cd khuta
```

### 2️⃣ Install dependencies  

```sh
flutter pub get
```

### 3️⃣ Run the app  

#### 📌 Development Flavor  

```sh

```

#### 📌 Production Flavor  

