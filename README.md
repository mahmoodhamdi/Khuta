# تطبيق "خُطى" - ADHD Assessment Application

## نظرة عامة
"خُطى" هو تطبيق رقمي مبتكر يهدف إلى تقييم وإدارة اضطراب فرط الحركة ونقص الانتباه (ADHD) لدى الأطفال باستخدام **مقياس كونرز** المعتمد. التطبيق يقدم أدوات لتقييم الأعراض وتحليلها وتقديم توصيات عملية لتحسين حياة الأطفال وأسرهم.

## المميزات الرئيسية
1. **مقياس كونرز**:
   - استبيان رقمي لتقييم الأعراض.
   - تحليل النتائج باستخدام الدرجات التائية.

2. **تحليل النتائج**:
   - تقديم تقارير مرئية واضحة.
   - توصيات علاجية مخصصة.

3. **نظام متابعة**:
   - مراقبة التقدم بمرور الوقت.
   - عرض المقارنات التاريخية.

4. **التواصل**:
   - ربط المستخدمين بالمتخصصين.
   - مشاركة التقارير بسهولة.

---

## التقنيات المستخدمة
- **Flutter** لإطار العمل.
- **SQLite** لتخزين البيانات محليًا.
- **Firebase** لمزامنة البيانات السحابية.

## الأمان والخصوصية
- **تشفير كامل** للبيانات الحساسة.
- **حقوق وصول محددة** بناءً على نوع المستخدم.

---

## الفئة المستهدفة
- الآباء والأمهات.
- المعلمون.
- المتخصصون النفسيون.

---

## بدء الاستخدام
1. تسجيل الدخول أو إنشاء حساب.
2. إضافة بيانات الطفل.
3. إجراء التقييم.
4. عرض النتائج ومشاركتها.
