# تطبيق "خُطى" - ADHD Assessment Application

## نظرة عامة

"خُطى" هو تطبيق رقمي مبتكر يهدف إلى تقييم وإدارة اضطراب فرط الحركة ونقص الانتباه (ADHD) لدى الأطفال باستخدام تقنيات متقدمة. التطبيق يقدم أدوات لتقييم الأعراض وتحليلها وتقديم توصيات عملية لتحسين حياة الأطفال وأسرهم.

## المميزات الرئيسية

1. **التقييم الذكي**:
   - مقياس كونرز الرقمي للتقييم
   - تحليل متقدم باستخدام Google Gemini AI
   - حساب الدرجات التائية أوتوماتيكياً

2. **إدارة المستخدمين**:
   - حسابات منفصلة للوالدين والمتخصصين
   - إدارة ملفات الأطفال
   - نظام صلاحيات متعدد المستويات

3. **المتابعة والتحليل**:
   - تقارير تفصيلية عن التقدم
   - توصيات ذكية مخصصة
   - تحليل بياني للنتائج

4. **التواصل والمشاركة**:
   - تواصل مباشر بين الوالدين والمتخصصين
   - مشاركة التقارير بصيغ متعددة
   - نظام إشعارات للمتابعة

---

## التقنيات المستخدمة

- **واجهة المستخدم**: Flutter/Dart
- **إدارة الحالة**: Provider & BLoC
- **قاعدة البيانات**:
  - SQLite (محلياً)
  - Firebase Firestore (سحابياً)
- **الذكاء الاصطناعي**: Google Gemini API
- **الأمان**: Firebase Authentication

## متطلبات النظام

- **نظام التشغيل**: iOS 13+ / Android 8.0+
- **الاتصال**: إنترنت للمزامنة والتحليل
- **المساحة**: 100MB كحد أدنى

## الأمان والخصوصية

- تشفير End-to-End للبيانات الحساسة
- تخزين محلي آمن للمعلومات الشخصية
- نظام صلاحيات متعدد المستويات
- مزامنة آمنة مع السحابة

## خطوات البدء

1. **التسجيل**:
   - إنشاء حساب جديد
   - اختيار نوع المستخدم (ولي أمر/متخصص)
   - تأكيد البريد الإلكتروني

2. **إعداد الملف**:
   - إضافة المعلومات الشخصية
   - إضافة معلومات الطفل (للوالدين)
   - ربط المتخصص (اختياري)

3. **بدء التقييم**:
   - اختيار الطفل
   - إكمال مقياس كونرز
   - مراجعة النتائج والتوصيات

4. **المتابعة**:
   - جدولة تقييمات دورية
   - متابعة التقدم
   - التواصل مع المتخصصين
