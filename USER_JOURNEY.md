# User Journey in "Khutwa" App (MVP)

## المراحل الرئيسية

## Main Stages

### 1. التسجيل والإعداد

### 1. Registration and Setup

#### نقطة البداية

#### Starting Point

- تحميل التطبيق وفتحه.
- Download and open the application.
- اختيار نوع الحساب (ولي أمر/معلم).
- Select account type (parent/teacher).
- تسجيل الدخول أو إنشاء حساب جديد.
- Login or create new account.

#### الإعداد

#### Setup

- إدخال بيانات الطفل (اسم، عمر، ملاحظات).
- Enter child's information (name, age, notes).
- ربط الطفل بمختص إذا لزم.
- Link child with specialist if needed.

### 2. إجراء التقييم

### 2. Conducting Assessment

#### اختيار التقييم

#### Selecting Assessment

- اختيار مقياس كونرز.
- Select Conners Scale.
- تحديد نوع التقييم (ولي أمر/معلم).
- Specify assessment type (parent/teacher).

#### الإجابة على الأسئلة

#### Answering Questions

- عرض أسئلة التقييم واحدة تلو الأخرى.
- Display assessment questions one by one.
- اختيار الإجابة المناسبة (0-3).
- Choose appropriate answer (0-3).

#### حفظ التقييم

#### Saving Assessment

- إرسال الإجابات لتحليلها.
- Submit answers for analysis.
- عرض رسالة تأكيد بإتمام التقييم.
- Display confirmation message upon completion.

### 3. عرض النتائج

### 3. Displaying Results

#### تحليل النتائج

#### Results Analysis

- عرض الدرجات الخام لكل فئة.
- Display raw scores for each category.
- تحويل الدرجات إلى درجات تائية.
- Convert scores to T-scores.
- تقديم تفسير لكل درجة.
- Provide interpretation for each score.

#### التوصيات

#### Recommendations

- عرض نصائح وتوصيات عملية.
- Display practical advice and recommendations.
- إنشاء خطة علاجية بناءً على النتائج.
- Create treatment plan based on results.

### 4. التواصل والمتابعة

### 4. Communication and Follow-up

#### التواصل

#### Communication

- إرسال ومشاركة النتائج مع المختص.
- Send and share results with specialist.
- استقبال توصيات إضافية.
- Receive additional recommendations.

#### المتابعة

#### Follow-up

- إعادة التقييم بعد فترة زمنية محددة.
- Reassess after a specified time period.
- مقارنة النتائج لتتبع التقدم.
- Compare results to track progress.
