# رحلة المستخدم في تطبيق "خُطى" (MVP)

## المراحل الرئيسية

### 1. التسجيل والإعداد

#### نقطة البداية
- تحميل التطبيق
- فتح التطبيق
- اختيار نوع الحساب

#### التسجيل
- اختيار نوع الحساب (والد/متخصص)
- إنشاء حساب جديد بالبريد الإلكتروني
- إدخال المعلومات الأساسية
- تأكيد الحساب

### 2. إعداد الملف الشخصي

#### حساب الوالدين
- إضافة معلومات الطفل الأساسية
- البحث عن متخصص

#### حساب المتخصص
- إضافة التخصص
- البدء في استقبال الحالات

### 3. التقييم الأولي

#### إجراء التقييم (الوالدين)
- اختيار مقياس كونرز
- الإجابة على الأسئلة
- إرسال التقييم للمتخصص

#### مراجعة التقييم (المتخصص)
- استلام نتائج التقييم
- تحليل الإجابات
- كتابة التوصيات الأساسية

### 4. التواصل والمتابعة

#### التواصل المباشر
- إرسال واستقبال الرسائل النصية
- متابعة حالة الطفل

---
ملاحظة: الميزات التالية ستضاف في التحديثات المستقبلية:
- نظام تقييم متقدم
- خطط علاج مفصلة
- جدولة مواعيد
- تقارير وإحصائيات متقدمة
- نظام إشعارات شامل
- مشاركة الملفات والتقارير
